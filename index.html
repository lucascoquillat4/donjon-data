Introduction

Dans un futur lointain, une équipe de chercheurs découvre qu'une histoire cruciale concernant un mystérieux donjon à l’Est de la Géorgie manque dans leurs archives. Pour reconstituer cette histoire, ils décident d'utiliser leur machine à voyager dans le temps, bien qu’elle soit si dégradée qu’un seul aller-retour est possible. Ils recrutent 3 spécialistes du passé, les Géorgiens y ajoutent également 3 experts. Les six aventuriers doivent s’aventurer dans ce donjon oublié, confrontant d’éventuels dangers, monstres ou mystères encore plus sombres.

Lancelot

Lancelot est un chevalier pour le compte du roi Tbilisi. Type Force.

Eldrin

C’est un mage enseignant en Institut International de Magie (IIM). Type Magie.

Ziggy

C’est le fou du roi, également chef d’une guilde secrète de chasseurs de monstres. Type Agilité.

Chronos

C’est un soldat à la retraite, enrôlé pour cette mission, il est très fort mais également très rapide. Type Force.

Professeur Nexus

C’est un chercheur donnant des cours d’histoires au centre d’Enseignement Scientifiques International Libre de Vinci (ESILV). Type Magie

C-3PU

C’est un robot d’assistance présent pour aider le professeur Nexus. Type Agilité.
